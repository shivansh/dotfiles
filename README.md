##Requirements
* zsh
* curl
* Powerline fonts
* Vim-Plug (for plugin management)
* neovim/vim

##Installation
* Run `bash env-setup.sh`
* For installing the vim plugins, just run `:PlugInstall` from inside your vim buffer.
